Test-CTXAPI_Header

SYNOPSIS

Checks that the connection is still valid, and the token hasn't expired.

SYNTAX

Test-CTXAPI_Header [[-APIHeader] <Object>] [-AutoRenew] [<CommonParameters>]

DESCRIPTION

Checks that the connection is still valid, and the token hasn't expired.

EXAMPLES

EXAMPLE 1

Test-CTXAPI_Header -APIHeader $APIHeader -AutoRenew

PARAMETERS
